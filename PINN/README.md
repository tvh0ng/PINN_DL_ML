### Real-World Applications
- **Material science**: Identify thermal conductivity, elastic moduli
- **Fluid mechanics**: Discover viscosity, permeability
- **Biology**: Estimate diffusion rates, reaction rates
- **Climate science**: Calibrate ocean/atmosphere models from sparse data

### Summary of All 4 Projects

| Project | PDE | Input | Key Concept |
|---------|-----|-------|-------------|
| 1. Heat Eq. | 1D steady-state | x | Basic PINN, autograd |
| 2. Burgers' | 1D time-dependent | (x, t) | Nonlinear PDE, IC/BC |
| 3. Poisson 2D | 2D steady-state | (x, y) | 2D domain, Laplacian |
| 4. Inverse | 1D diffusion | (x, t) + data | Parameter identification |
